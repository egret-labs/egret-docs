Egret Engine 3.0 包含了白鹭时代研发的遵循HTML5标准的2D引擎及全新打造的3D引擎，它解决了HTML5性能问题及碎片化问题，灵活地满足开发者开发2D或3D游戏的需求，并有着极强的跨平台运行能力。

下面介绍 Egret Engine 3.0.3 到 Egret Engine 3.0.4 之间的更新详情。


### Egret Engine 3D
